What is Loot? What is Deck? Where is the game? Can I join the project? What to expect in the future? Loot is a collection of 8,000 unique bags of adventuring gear, originally released by Dom Hofmann. Each loot bag contains 8 items: a piece for an adventurer's chest, foot, hand, head, neck, ring, waist, and weapon. Go to www.lootproject.com for more information. Deck is a project to create a TCG or Trading Card Game on top of Loot. It’s a small passion project started by @FrappeMango in hopes to be able to create a Web3 integrated Multiplayer game. The project focuses on creating cards based on data / lore from Loot and creatinga game where you can use these cards. As Deck is still a WIP project, the game isn’t finished as of writing. However, to show progress the latest version of the game is in the “Game” tab of this website. Yes, if you are able to Code, Draw, Write, Build or just have a cool idea, feel free to contact @FrappeMango on Twitter. Let’s build this project together!
